POS RECEIPT EDITOR

This module help user to create/edit receipt template from Odoo backend

Create / edit receipt template

Go to Point of Sale → Configuration → Receipt template

  • Specify Template name
  • Specify Type
  • Specify Template
    •

Add receipt template in Pos configuration

Go to Point of Sale → Configuration → Point of sale. Go to Bill & Receipt section; check Use custom ticket
