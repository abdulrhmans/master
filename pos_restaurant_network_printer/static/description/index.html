ESC/POS Network printer configuration for Pos Restaurant

This module work ONLY with ESC/POS Network Printer. You don't need POS Box

Add orders printer

Go to Point of Sale → Configuration → Order Printers

  • Specify Printer Name
  • Specify IP Address of network printer
  • Specify Port of network printer. Default : 9100
  • Specify Printed Product Categories
    •

Add network printer in Pos configuration

Go to Point of Sale → Configuration → Point of sale. Edit the Pos and add printer in Oders printers section

Set bill print mode to Network

Go to Point of Sale → Configuration → Point of sale. Edit the Pos and set the bill print mode
